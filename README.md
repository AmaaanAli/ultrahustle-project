## Setup Instructions

1. **Install Dependencies**
   Ensure you have Node.js installed. Open your terminal in the root directory of the project and run:
   ```bash
   npm install
   ```

2. **Start the Development Server**
   Start the blazing-fast local Vite server with:
   ```bash
   npm run dev
   ```

3. **View the Application**
   Open your browser and navigate to `http://localhost:5173/` (or whichever port is specified in your terminal output) to view the dashboard live!
